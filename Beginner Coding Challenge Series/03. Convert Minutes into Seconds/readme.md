<h1 align='center'>~ Convert Minutes into Seconds ~</h1>

<p>Write a function that takes an integer minutes and converts it to seconds.</p>

<h3>Examples</h3>

```
convert(5) ➞ 300
convert(3) ➞ 180
convert(2) ➞ 120
```
