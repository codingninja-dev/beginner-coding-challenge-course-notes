<h1 align='center'>~ Less Than 100? ~</h1>

<p>Given two numbers, return true if the sum of both numbers is less than 100. Otherwise return false.</p>

<h3>Examples</h3>

```
lessThan100(22, 15) ➞ true
// 22 + 15 = 37
lessThan100(83, 34) ➞ false
// 83 + 34 = 117
lessThan100(3, 77) ➞ true
```
