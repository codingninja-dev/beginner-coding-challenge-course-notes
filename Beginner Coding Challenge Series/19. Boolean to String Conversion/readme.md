<h1 align='center'>~ Boolean to String Conversion ~</h1>

<p>Create a function that takes a boolean variable flag and returns it as a string.</p>

<h3>Examples</h3>

```
boolToString(true) ➞ "true"
boolToString(false) ➞ "false"
```
