<h1 align='center'>~ Power Calculator ~</h1>

<p>Create a function that takes voltage and current and returns the calculated power.</p>

<h3>Examples</h3>

```
circuitPower(230, 10) ➞ 2300
circuitPower(110, 3) ➞ 330
circuitPower(480, 20) ➞ 9600
```
