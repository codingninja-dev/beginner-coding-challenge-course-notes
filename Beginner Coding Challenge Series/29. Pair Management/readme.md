<h1 align='center'>~ Pair Management ~</h1>

<p>Given two arguments, return an array which contains these two arguments.</p>

<h3>Examples</h3>

```
makePair(1, 2) ➞ [1, 2]
makePair(51, 21) ➞ [51, 21]
makePair(512124, 215) ➞ [512124, 215]
```
