<h1 align='center'>~ Return the Sum of Two Numbers ~</h1>

<p>Create a function that takes two numbers as arguments and return their sum.</p>

<h3>Examples</h3>

```
addition(3, 2) ➞ 5
addition(-3, -6) ➞ -9
addition(7, 3) ➞ 10
```
