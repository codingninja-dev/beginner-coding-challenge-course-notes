<h1 align='center'>~ Two Makes Ten ~</h1>

<p>Create a function that takes two arguments. Both arguments are integers, a and b. Return true if one of them is 10 or if their sum is 10.</p>

<h3>Examples</h3>

```
makesTen(9, 10) ➞ true
makesTen(9, 9) ➞ false
makesTen(1, 9) ➞ true
```
