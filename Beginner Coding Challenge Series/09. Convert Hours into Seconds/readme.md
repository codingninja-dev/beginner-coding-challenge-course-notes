<h1 align='center'>~ Convert Hours into Seconds ~</h1>

<p>Write a function that converts hours into seconds.</p>

<h3>Examples</h3>

```
howManySeconds(2) ➞ 7200
howManySeconds(10) ➞ 36000
howManySeconds(24) ➞ 86400
```
