<h1 align='center'>~ Return Negative ~</h1>

<p>Create a function that takes a number as an argument and returns negative of that number. Return negative numbers without any change.</p>

<h3>Examples</h3>

```
returnNegative(4) ➞ -4
returnNegative(15) ➞ -15
returnNegative(-4) ➞ -4
returnNegative(0) ➞ 0
```
