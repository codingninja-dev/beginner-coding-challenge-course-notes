<h1 align='center'>~ Convert Hours and Minutes into Seconds ~</h1>

<p>Write a function that takes two integers (hours, minutes), converts them to seconds, and adds them.</p>

<h3>Examples</h3>

```
convert(1, 3) ➞ 3780
convert(2, 0) ➞ 7200
convert(0, 0) ➞ 0
```
